# 📝 Academic Note Pad

## 📌 Project Description

This Figma project showcases a clean and organized mobile app design for taking and managing academic notes. Students can create notes, organize subjects, set reminders, and keep track of their study materials in one place.

### ✨ Features

* 🏠 Home Dashboard
* 📝 Create & Edit Notes
* 📚 Subject Categories
* ⏰ Study Reminders
* 👤 User Profile

### 🛠️ Tools Used

* Figma

### 🎯 Purpose

To design a simple, user-friendly, and visually appealing note-taking application that helps students stay organized and improve their study productivity.
